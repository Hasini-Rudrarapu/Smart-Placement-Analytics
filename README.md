# 🚀 Smart Placement Analytics System

## 📌 Overview
Streamlit-based analytics project to analyze student placement trends and insights.

## 🎯 Features
- Placement dashboard
- Branch analysis
- Skills vs placement
- Salary insights
- Readiness score

## 🛠️ Tech Stack
Python, Streamlit, Pandas, Plotly, Scikit-learn
